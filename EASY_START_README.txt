EASY WINDOWS START

1. Install Node.js if you do not already have it:
   https://nodejs.org

2. Double-click:
   START_GAME_WINDOWS.bat

3. Wait for it to open your browser.

4. The game opens at:
   http://localhost:3000

TEST MULTIPLAYER ON ONE COMPUTER

- Open one browser tab and create a room.
- Open a second browser tab at http://localhost:3000
- Join the same room code.
- Start the match from the first tab.

IMPORTANT

Your friends cannot join your localhost link from their house.
For real friends online, you still need to host the server online.
